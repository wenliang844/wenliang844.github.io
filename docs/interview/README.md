# 面试项目素材：智能视频分析预警平台

本目录基于项目 `D:\java\lianqian\code\code-for-vue\wsdc-znfx` 生成，用于面试前准备和简历项目包装。

## 文档目录

1. [项目深度分析文档](./01-项目深度分析文档.md)  
   用于系统理解项目背景、架构、模块、链路、技术实现、性能优化、缓存设计和优化方向。

2. [简历项目经历](./02-简历项目经历.md)  
   按 Java 后端、AI 工程化、规则引擎、政企业务域等方向提供简历写法。

3. [面试讲解稿](./03-面试讲解稿.md)  
   提供 3 分钟、5 分钟、10 分钟三个版本，可以直接背诵或改成自己的表达。

4. [面试问答库](./04-面试问答库.md)  
   覆盖架构、Spring Boot、MyBatis、Redis、MQ、Dubbo/Nacos、ES、事务、性能、可靠性、扩展性、代码质量等问题。

5. [项目亮点包装](./05-项目亮点包装.md)  
   提炼 15 个面试可讲亮点，每个亮点包含业务价值、技术实现、可追问点和回答思路。

## 推荐准备顺序

1. 先读 `03-面试讲解稿.md` 的 3 分钟版本，形成主线。
2. 再读 `01-项目深度分析文档.md` 的核心业务流程和技术实现。
3. 背熟 `05-项目亮点包装.md` 的前 5 个亮点。
4. 最后用 `04-面试问答库.md` 做模拟追问。
